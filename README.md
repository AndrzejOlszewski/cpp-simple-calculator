# Simple C++ Calculator

A lightweight, console-based calculator application written in C++. This project was built entirely from memory to practice fundamental programming concepts.

## 🚀 Features
* **Basic Math**: Supports addition, subtraction, multiplication, and division.
* **Flow Control**: Implements `switch-case` logic for fast operator routing.
* **Error Handling**: Detects and prevents division-by-zero crashes.
* **Input Validation**: Rejects invalid arithmetic operators gracefully.

## 💻 Concepts Covered
* Standard input/output streams (`cin`, `cout`).
* Variables and data types (`double`, `char`).
* Preventing switch "fall-through" using `break`.

## 🛠️ How to Run
1. Open your terminal or command prompt.
2. Compile the file using a C++ compiler:
   ```bash
   g++ main.cpp -o calculator
   ```
3. Run the compiled program:
   ```bash
   ./calculator
   ```
