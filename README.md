# Simple C++ Calculator

A lightweight console-based calculator application written in C++.  
This project was created to practice fundamental C++ programming concepts, including control flow, user input, and basic error handling.

## 🚀 Features

- **Basic Math:** Supports addition, subtraction, multiplication, and division.
- **Operator Selection:** Uses `switch-case` logic to handle different operations.
- **Error Handling:** Prevents division-by-zero errors.
- **Input Validation:** Handles invalid operators safely.

## 💻 Concepts Covered

- Standard input/output streams (`cin`, `cout`)
- Variables and data types (`double`, `char`)
- `switch` statements and `break`
- Basic error handling

## 🛠️ How to Run

1. Open your terminal or command prompt.
2. Compile the program:

```bash
g++ main.cpp -o calculator
